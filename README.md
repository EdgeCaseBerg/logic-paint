Logicross
===============================

This is a silly little picross/logicpaint/griddler game that I made
because I really enjoyed the [miku game] of a similar name. I toiled
away at it for a month or so while exploring the use of [egor] as a
gaming engine library as my first foray into using rust rather than
Java for game dev hobby coding.

The entire experience, stream of consciousness, and whatever you'd like
to refer to a dev journal as is all cataloged on my website over here:

1. [the entire game post]
2. [creating the solver in the level editor]

So if you want to know how this was made, then settle in with a cup
of warm beverage and blanket and read those two posts.
